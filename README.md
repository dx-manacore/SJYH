# نظام إدارة المطابع
## Printing Management System

نظام شامل لإدارة المطابع يتضمن:
- 📄 إدارة الفواتير (المبيعات والمشتريات)
- 📦 إدارة المخزون
- 🏭 إدارة المنتجات
- 👥 إدارة الموارد البشرية
- 🔐 نظام الصلاحيات والأدوار
- 🏢 إدارة الأقسام

### التقنيات المستخدمة
- **Python 3.9+**
- **Django** - إطار العمل الرئيسي
- **Django REST Framework** - للـ API
- **PostgreSQL** - قاعدة البيانات
- **Celery** - لمعالجة المهام غير المتزامنة
- **Redis** - للتخزين المؤقت والرسائل

### البنية
```
printing-management-system/
├── config/              # إعدادات المشروع
├── apps/
│   ├── invoices/       # نظام الفواتير
│   ├── inventory/      # إدارة المخزون
│   ├── products/       # إدارة المنتجات
│   ├── hr/             # الموارد البشرية
│   ├── permissions/    # نظام الصلاحيات
│   └── departments/    # إدارة الأقسام
├── tests/              # الاختبارات
└── docs/               # التوثيق
```
